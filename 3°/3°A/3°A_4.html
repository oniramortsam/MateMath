le funzioni elementari

\[\tan\left(\alpha\right)=\frac{\sin\left(\alpha\right)}{\cos\left(\alpha\right)}\qquad\cot\left(\alpha\right)=\frac{\cos\left(\alpha\right)}{\sin\left(\alpha\right)}\] Seguono le formule di addizione:
  • \[\tan\left(\alpha+\beta\right)=\frac{\tan\left(\alpha\right)+\tan\left(\beta\right)}{1-\tan\left(\alpha\right)\cdot\tan\left(\beta\right)}\]
  • \[\cot\left(\alpha+\beta\right)=\frac{\cot\left(\alpha\right)\cdot\cot\left(\beta\right)-1}{\cot\left(\beta\right)+\cot\left(\alpha\right)}\]
  • \[\tan\left(\alpha-\beta\right)=\frac{\tan\left(\alpha\right)-\tan\left(\beta\right)}{1+\tan\left(\alpha\right)\cdot\tan\left(\beta\right)}\]
  • \[\cot\left(\alpha-\beta\right)=\frac{\cot\left(\alpha\right)\cdot\cot\left(\beta\right)+1}{\cot\left(\beta\right)-\cot\left(\alpha\right)}\]
Seguono le formule di werner:
  • \[\cot\left(\alpha\right)\cdot\cot\left(\beta\right)=\frac{\cos\left(\alpha-\beta\right)+\cos\left(\alpha+\beta\right)}{\cos\left(\alpha-\beta\right)-\cos\left(\alpha+\beta\right)}\]
  • \[\cos\left(\alpha\right)\cdot\sin\left(\beta\right)=\frac{\sin\left(\alpha+\beta\right)-\sin\left(\alpha-\beta\right)}{2}\]
  • \[\sin\left(\alpha\right)\cdot\cos\left(\beta\right)=\frac{\sin\left(\alpha+\beta\right)+\sin\left(\alpha-\beta\right)}{2}\]
  • \[\tan\left(\alpha\right)\cdot\tan\left(\beta\right)=\frac{\cos\left(\alpha-\beta\right)-\cos\left(\alpha+\beta\right)}{\cos\left(\alpha-\beta\right)+\cos\left(\alpha+\beta\right)}\]
Seguono le formule di prostaferesi:
  • \[\cos\left(\beta\right)+\cos\left(\alpha\right)=2\cdot\cos\left(\frac{\alpha+\beta}{2}\right)\cdot\cos\left(\frac{\alpha-\beta}{2}\right)\]
  • \[\sin\left(\alpha\right)-\sin\left(\beta\right)=2\cdot\cos\left(\frac{\alpha+\beta}{2}\right)\cdot\sin\left(\frac{\alpha-\beta}{2}\right)\]
  • \[\sin\left(\alpha\right)+\sin\left(\beta\right)=2\cdot\sin\left(\frac{\alpha+\beta}{2}\right)\cdot\cos\left(\frac{\alpha-\beta}{2}\right)\]
  • \[\cos\left(\beta\right)-\cos\left(\alpha\right)=2\cdot\sin\left(\frac{\alpha+\beta}{2}\right)\cdot\sin\left(\frac{\alpha-\beta}{2}\right)\]
\[\sinh\left(\alpha\right)=\frac{\exp\left(2\cdot\alpha\right)-1}{2\cdot\exp\left(\alpha\right)}\qquad\coth\left(\alpha\right)=\frac{\exp\left(2\cdot\alpha\right)}{2\cdot\exp\left(\alpha\right)}\] \[\tanh\left(\alpha\right)=\frac{\sinh\left(\alpha\right)}{\cosh\left(\alpha\right)}\qquad\coth\left(\alpha\right)=\frac{\cosh\left(\alpha\right)}{\sinh\left(\alpha\right)}\]
