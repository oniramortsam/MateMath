Una relazione d'ordine soddisfa le proprietà:
  • Simmetrica: \[\forall\left({\rm x},{\rm y}\right)\in{\rm A}\times{\rm A}\quad{\cal R}\left({\rm x},{\rm y}\right)\land{\cal R}\left({\rm y},{\rm x}\right)\Longrightarrow{\rm x}={\rm y}\]
  • Transitiva: \[\forall\left({\rm x},{\rm y},{\rm z}\right)\in{\rm A}\times{\rm A}\times{\rm A}\quad{\cal R}\left({\rm x},{\rm y}\right)\land{\cal R}\left({\rm y},{\rm z}\right)\Longrightarrow{\cal R}\left({\rm x},{\rm z}\right)\]
Si può dire totale se: \[\forall\left({\rm x},{\rm y}\right)\in{\rm A}\times{\rm A}\quad{\cal R}\left({\rm x},{\rm y}\right)\lor{\cal R}\left({\rm y},{\rm x}\right)\]
Per una relazione d'ordine:
  • si dice minorante di un sottoinsieme
  • si dice maggiorante di un sottoinsieme
  • si dice minimo di un sottoinsieme
  • si dice massimo di un sottoinsieme
  • si dice estremo inferiore di un sottoinsieme
  • si dice estremo superiore di un sottoinsieme
Una relazione d'ordine si dice ben fondata se ogni sottoinsieme ammette minimale.
